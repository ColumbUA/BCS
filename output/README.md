# Управління ротою РРР — пакет файлів

Згенеровано на основі **БЧС роти радіо та радіотехнічної розвідки розвідувального батальйону** станом на **02.05.2026** (109 осіб).

---

## 1. Файли MS Project (.xml)

Відкриваються через **File → Open → Type: XML** у MS Project 2013+ (включно з 2016/2019/365).

### `Управління ротою РРР - Організаційна структура.xml`
WBS-ієрархія підрозділів роти як завдання + особовий склад як ресурси.
- **125 завдань** (Рота → Управління / Група / Взводи / Майстерня → Відділення → Посади)
- **23 ресурси** — особовий склад з ПІБ, позивними, званнями
- **23 призначення** — кожна посада прив'язана до своєї особи

Структура:
| WBS | Підрозділ | Осіб |
|-----|-----------|------|
| 1.1 | Управління роти | 7 |
| 1.2 | Група обробки інформації | 8 |
| 1.3 | 1 Взвод радіорозвідки | 20 |
| 1.4 | 2 Взвод радіорозвідки | 29 |
| 1.5 | Взвод радіоелектронної розвідки | 26 |
| 1.6 | Взвод безпілотних авіаційних комплексів | 16 |
| 1.7 | Ремонтна майстерня озброєння | 3 |
| **Разом** | | **109** |

### `Управління ротою РРР - Бойове управління.xml`
Цикл управління ротою (24 год) — діаграма Ганта.
- **38 завдань** у 7 фазах із залежностями (Finish-to-Start)
- **12 ресурсів** — ключові посадові особи (КР, ЗКР з ПП, КВ-1..4, начальники, сержанти)
- **52 призначення**

Фази:
1. Підготовка управління (отримання БЗ, з'ясування, орієнтування, розрахунок часу)
2. Оцінка обстановки (противник, свої сили, район, час, тил)
3. Прийняття рішення (формування задуму, доповідь КБ)
4. Постановка задач підрозділам (наказ + 6 розпоряджень)
5. Забезпечення та взаємодія (зв'язок, тил, медицина, психологія)
6. Контроль і виконання (готовність, виконання БЗ, коригування)
7. Доповіді та підсумки (поточні + підсумкова + наст. доба)

Календар: **24/7** (бойовий режим).

## 3. Як відкрити XML у MS Project

1. Запустіть **Microsoft Project** (2013 або новіша)
2. **File → Open**
3. У діалозі виберіть тип файлу **XML Format (\*.xml)**
4. Виберіть потрібний `.xml` файл → відкриється як новий проєкт

> **Примітка:** при першому відкритті MS Project може запитати схему збереження (Schema). Натисніть **OK** — імпорт пройде з повним збереженням ієрархії, ресурсів і призначень.

Файли згенеровані скриптами в `/app/scripts/`:

- `gen_blockschemes.py` — генератор SVG-схем

Для оновлення замініть Excel-файл і повторно запустіть скрипти.
